App control de stock

App de control de stock en Python, permite controlar el stock de productos de la siguiente manera: Agregar nuevo producto: Asignando Nombre, Precio, Stock, Sección a la que pertenece el producto y una imagen. Modificando el producto ya creado y Eliminar el producto. (un crud) También cuenta con un Login donde el usuario puede darse de alta. La app esta diseñada y desarrollada con HTML5, CSS3, Bootstrap 4, JQuery, Django y SQLite3. Contiene archivos estáticos de Bootstrap y JQuery para acelerar la carga de las mismas. Tiene un backend soportado por Django 2.2.7 que aporta carga de templates dinámicamente además de aportar un administrador de usuario para la carga de nuevos usuarios y stock de productos con BBdd. Python 3.7 by Django 2.2.7
